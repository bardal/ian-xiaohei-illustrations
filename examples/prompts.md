# Prompt Examples

The following prompts can be copied directly into Codex.

## Shot list only (no image generation)

```text
Use $ian-xiaohei-illustrations — do not generate images yet.
Analyze the article below and identify which sections deserve an illustration. Output a shot list of about 5 images.
For each image, specify:
- Which paragraph it follows
- The image theme
- The core meaning
- The structure type
- What Xiaohei is doing in the image
- Suggested elements
- Suggested annotation labels

<paste article here>
```

## Article illustrations

```text
Use $ian-xiaohei-illustrations to generate 4 Xiaohei absurd article illustrations for the article below.
Requirements: 16:9 horizontal, pure white background, black hand-drawn line art, sparse red/orange/blue handwritten annotations.
Each image communicates only one core structure; no PPT infographics, no cute cartoons.

<paste article here>
```

## Long-article illustration strategy

```text
Use $ian-xiaohei-illustrations to create an illustration strategy for this long article.
Do not distribute images evenly — only pick cognitive anchors: core judgments, input-output loops, before/after contrasts, common pitfalls, handoff paths.
Default to 6-8 images; output the shot list only, do not generate images.

<paste article here>
```

## Single concept — one image

```text
Use $ian-xiaohei-illustrations to generate one 16:9 article illustration for this idea:

Trust is not proclaimed — it is laid down one piece of evidence at a time.

The image should be absurd but clean. Xiaohei must perform the core action.
Use at most 5 annotation labels, keep them short.
```

## Workflow theme

```text
Use $ian-xiaohei-illustrations to generate an image for "turning one piece of raw material into three types of content: traffic, trust, and conversion."
Do not draw a formal flowchart; do not copy the old "one fish many uses" example.
Invent a new low-tech metaphor and give Xiaohei the core action.
```

## Edit image — remove a title

```text
Use $ian-xiaohei-illustrations to edit this image.
Remove the "Workflow / flowchart" title and underline in the top-left corner; keep everything else unchanged.
Do not add any new text or objects.
```

## Edit image — strengthen Xiaohei's presence

```text
Use $ian-xiaohei-illustrations — this image has the right direction, but Xiaohei feels decorative.
Keep the core meaning; regenerate a version where Xiaohei is the one actually driving the structure.
Make the image a bit more absurd, but still pure white, clean, and minimal text.
```

## Generate a style sample set

```text
Use $ian-xiaohei-illustrations to output 5 Xiaohei article illustration concepts on different themes.
Themes: information overload, product validation, content compounding, solo business, trust building.
Generate each image separately; do not combine them into one.
```
